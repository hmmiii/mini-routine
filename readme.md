# 항해99 18기 사전스터디 1조 토이프로젝트 '미니루틴'

## 📌 프로젝트 소개

항해99 18기 사전스터디 1조의 파이썬 Flask 토이 프로젝트 '미니루틴' 입니다! 개인 루틴(습관)을 추가하고 체크할 수 있는 투두리스트 웹앱입니다.

구현된 기능은 다음과 같습니다.

- 회원 가입
- 로그인
- 루틴 조회
- 루틴 등록
- 루틴 수정
- 루틴 삭제
- 루틴 체크
- 서버 가동시 루틴 체크 초기화
